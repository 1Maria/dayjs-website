Contributing

这篇文档描述了如何为 dayjs 及社区作出贡献。

遇到问题 📋

如果你遇到了任何问题，我们建议你前往 dayjs/issues 提出并且把它描述出来。

我们将尽快帮助你。

想要添加一些代码？ 😌 (Pull a request)

我们也建议您先在 dayjs/issues 描述它。

这将帮助所有 dayjs 成员能够方便地与您讨论。
