DAYJS ORGANIZATION

⏰ Fast 2KB immutable date library alternative to Moment.js with the same modern API.

🤟 It can be used not only in JavaScript but also supports more languages!

GET STARTEDREAD THE DOCS

👀 Easy To Use

Day.js is a minimalist JavaScript library that parses, validates, manipulates, and displays dates and times for modern browsers with a largely Moment.js-compatible API.

If you use Moment.js, you already know how to use Day.js.

💪 Immutable & 🔥 Chainable

With immutable and chainable support, you can safely and easily to modify the existing dayjs instance in Day.js.

All operations will be intuitive.

Without any pains from now!

⚙️ Powerful Plugin Systems

Want more functionalities? The Plugin Systems will take you flying!

A plugin is an independent module that can be added to Day.js to extend functionality or add new features.

🌈 Multiple Languages Support

Not only in JavaScript!

You can also use Day.js in other languages. Like day.dart, day.rs (In Rust), day.clj (In Clojure)...

We will try our best to support more languages in the future.

🌐 I18n Support

Day.js has great support for internationalization.

But none of them will be included in your build unless you use it.
